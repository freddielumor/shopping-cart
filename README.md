# Shopping Cart

## How To Run The App:

```sh
$ npm install
```

```sh
$ npm start
```

```sh
$ npm test
```

```sh
$ npm run cy:open
```


## App Features:

###  Item List Page

- Add products to cart
- Track quantities and total cost
- Remove products from cart
- Max count validation
- Navigate to checkout

### Checkout Page

- Update product count
- Checkout
